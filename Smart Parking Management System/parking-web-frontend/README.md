# 智能停车场管理系统 - 纯前端版本

## 项目说明
这是一个纯前端实现的智能停车场管理系统，使用HTML、CSS、JavaScript开发，数据存储在浏览器的localStorage中。

## 功能特性
- 车位状态实时显示（空闲/占用）
- 车辆入场/出场管理
- 停车费用自动计算
- 停车记录查询
- 管理员登录和收费规则设置

## 技术栈
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (原生)
- localStorage (数据存储)

## 使用方法
1. 直接在浏览器中打开index.html文件
2. 默认管理员账户：admin/admin
3. 系统会自动初始化20个车位和基础收费规则
